# Phase 9.1: Design Change - Context

**Gathered:** 2026-02-06
**Status:** Ready for planning

<domain>
## Phase Boundary

Visual refresh and layout changes to align with KoorFameus brand identity. Derive visual system from existing logo. Does not include new pages, features, or dark mode (deferred).

</domain>

<decisions>
## Implementation Decisions

### Color Palette
- Primary color: Vibrant purple (violet) — creative, musical association
- Accent color: Orange — bold complementary pairing, child-friendly
- Replace current coral (#F97316) with new purple-orange palette
- Backgrounds: Warm off-white (cream/ivory tones), softer feel

### Typography & Spacing
- Font direction: More elegant to match choir/musical theme
- Content density: Balanced — moderate spacing, good information density
- Heading style: Subtle & elegant — understated, content-focused

### Visual Elements
- Card/component style: Soft shadows — subtle elevation, floating feel
- Corner rounding: Moderately rounded (8-12px)
- Images: Full bleed — extend to edges, immersive
- Photo style: Match new elegant aesthetic

### Claude's Discretion
- Exact purple shade (derive from logo if possible)
- Font pairing (elegant approach — could be serif+sans, refined sans, or display font for titles)
- Purple usage balance (accents vs bold sections vs gradients)
- Button styles (solid vs outlined hierarchy)
- Specific shadow values and elevation levels

</decisions>

<specifics>
## Specific Ideas

- "Cool & musical" direction — blues/purples for elegant, musical association
- Purple + orange complementary pairing suggested by user
- Current coral feels like brand mismatch
- Logo exists — derive visual system from it where possible
- Children's choir context: needs to feel approachable while elegant

</specifics>

<deferred>
## Deferred Ideas

- Dark mode support — add in future phase

</deferred>
