# Plan 01-02: Install and Configure shadcn/ui

## Objective
Install and configure shadcn/ui with the default style and neutral base color, then add essential UI components (Button, Card) that will be used in the placeholder page.

## must_haves
- shadcn/ui initialized with default style and CSS variables
- Button component installed with coral color variants
- Card component installed for content sections
- Components use the Koor Fameus color palette
- All components are accessible and responsive

## Tasks

<task id="1">
Initialize shadcn/ui in the project with default configuration.

```bash
cd c:/projects/KoorFameus
npx shadcn@latest init --defaults --yes
```

This will create:
- `components.json` configuration file
- `lib/utils.ts` utility functions
- Update `app/globals.css` with CSS variables
</task>

<task id="2" depends_on="1">
Update `components.json` to ensure proper configuration for the project.

The file should have these settings (verify or update):

```json
{
  "$schema": "https://ui.shadcn.com/schema.json",
  "style": "default",
  "rsc": true,
  "tsx": true,
  "tailwind": {
    "config": "tailwind.config.ts",
    "css": "app/globals.css",
    "baseColor": "neutral",
    "cssVariables": true,
    "prefix": ""
  },
  "aliases": {
    "components": "@/components",
    "utils": "@/lib/utils",
    "ui": "@/components/ui",
    "lib": "@/lib",
    "hooks": "@/hooks"
  },
  "iconLibrary": "lucide"
}
```
</task>

<task id="5" depends_on="3">
Update `components/ui/button.tsx` to add coral color variants.

Find the `buttonVariants` definition and add a coral variant to the `variant` object. Add after the existing variants:

```tsx
// In the variants object, add:
coral: "bg-coral text-white hover:bg-coral-dark shadow-sm transition-colors duration-200",
"coral-outline": "border-2 border-coral text-coral hover:bg-coral hover:text-white transition-colors duration-200",
```

The full buttonVariants should include both the default variants AND these new coral variants.
</task>

<task id="6" depends_on="1">
Update `app/globals.css` to merge shadcn/ui CSS variables with Koor Fameus design tokens.

Ensure the file contains both the @theme block (from 01-01) AND the shadcn/ui layer styles. The final file should look like:

```css
@import "tailwindcss";

@theme {
  /* Koor Fameus Color Palette */
  --color-coral: #F3504C;
  --color-coral-dark: #D34153;
  --color-coral-light: #F8807D;

  /* Text Colors */
  --color-text-primary: #1F1F1F;
  --color-text-secondary: #4A4A4A;
  --color-text-muted: #6B7280;

  /* Background Colors */
  --color-bg-white: #FFFFFF;
  --color-bg-section: #F4F4F4;
  --color-bg-accent: #FEF2F2;

  /* Design Tokens - Border Radius */
  --radius-sm: 0.375rem;
  --radius-md: 0.5rem;
  --radius-lg: 0.75rem;
  --radius-xl: 1rem;
  --radius-2xl: 1.5rem;

  /* Design Tokens - Shadows */
  --shadow-sm: 0 1px 2px 0 rgb(0 0 0 / 0.05);
  --shadow-md: 0 4px 6px -1px rgb(0 0 0 / 0.1), 0 2px 4px -2px rgb(0 0 0 / 0.1);
  --shadow-lg: 0 10px 15px -3px rgb(0 0 0 / 0.1), 0 4px 6px -4px rgb(0 0 0 / 0.1);

  /* Design Tokens - Transitions */
  --transition-fast: 150ms;
  --transition-normal: 200ms;
  --transition-slow: 300ms;

  /* Design Tokens - Spacing for sections */
  --spacing-section-sm: 3rem;
  --spacing-section-md: 4rem;
  --spacing-section-lg: 5rem;
}

@layer base {
  :root {
    --background: 0 0% 100%;
    --foreground: 0 0% 12%;
    --card: 0 0% 100%;
    --card-foreground: 0 0% 12%;
    --popover: 0 0% 100%;
    --popover-foreground: 0 0% 12%;
    --primary: 2 87% 63%;
    --primary-foreground: 0 0% 100%;
    --secondary: 0 0% 96%;
    --secondary-foreground: 0 0% 12%;
    --muted: 0 0% 96%;
    --muted-foreground: 0 0% 42%;
    --accent: 0 0% 96%;
    --accent-foreground: 0 0% 12%;
    --destructive: 0 84% 60%;
    --destructive-foreground: 0 0% 100%;
    --border: 0 0% 90%;
    --input: 0 0% 90%;
    --ring: 2 87% 63%;
    --radius: 0.75rem;
  }
}

@layer base {
  * {
    @apply border-border;
  }
  body {
    @apply bg-background text-foreground;
    font-family: var(--font-geist-sans), system-ui, sans-serif;
  }
}

/* Smooth scrolling */
html {
  scroll-behavior: smooth;
}

/* Focus styles for accessibility */
*:focus-visible {
  outline: 2px solid var(--color-coral);
  outline-offset: 2px;
}
```

Note: The `--primary` HSL value (2 87% 63%) corresponds to the coral color #F3504C.
</task>
