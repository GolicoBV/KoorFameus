# Plan 01-01: Initialize Next.js with Tailwind v4 and Geist Font

## Objective
Create a new Next.js 15 project with TypeScript, Tailwind CSS v4 configured with the Koor Fameus color palette, and Geist Sans font as the primary typeface.

## must_haves
- Next.js project initialized with App Router and TypeScript
- Tailwind CSS v4 configured with custom color tokens (koraalrood #F3504C, donkerrood #D34153, text #1F1F1F, background #FFFFFF, section #F4F4F4)
- Geist Sans font loaded and set as default font
- Project runs locally without errors
- Mobile-first responsive setup ready

## Tasks

<task id="1">
Initialize a new Next.js project with TypeScript and Tailwind CSS.

```bash
cd c:/projects/KoorFameus
npx create-next-app@latest . --typescript --tailwind --eslint --app --src-dir=false --import-alias="@/*" --turbopack --yes
```

Note: Using `--yes` to accept defaults. The `.` installs in current directory.
</task>

<task id="2" depends_on="1">
Install Geist font package for Next.js.


</task>

<task id="3" depends_on="2">
Update `app/layout.tsx` to configure Geist Sans as the primary font.

Replace the entire file content with:

```tsx
import type { Metadata } from "next";
import { GeistSans } from "geist/font/sans";
import "./globals.css";

export const metadata: Metadata = {
  title: "Koor Fameus - Kinderkoor",
  description: "Welkom bij Koor Fameus, een energiek kinderkoor",
};

export default function RootLayout({
  children,
}: Readonly<{
  children: React.ReactNode;
}>) {
  return (
    <html lang="nl" className={GeistSans.className}>
      <body className="antialiased bg-white text-gray-900">
        {children}
      </body>
    </html>
  );
}
```
</task>

<task id="4" depends_on="1">
Update `app/globals.css` to configure Tailwind v4 with custom design tokens using @theme.

Replace the entire file content with:

```css
@import "tailwindcss";

@theme {
  /* Koor Fameus Color Palette */
  --color-coral: #F3504C;
  --color-coral-dark: #D34153;
  --color-coral-light: #F8807D;

  /* Text Colors */
  --color-text-primary: #1F1F1F;
  --color-text-secondary: #4A4A4A;
  --color-text-muted: #6B7280;

  /* Background Colors */
  --color-bg-white: #FFFFFF;
  --color-bg-section: #F4F4F4;
  --color-bg-accent: #FEF2F2;

  /* Design Tokens - Border Radius */
  --radius-sm: 0.375rem;
  --radius-md: 0.5rem;
  --radius-lg: 0.75rem;
  --radius-xl: 1rem;
  --radius-2xl: 1.5rem;

  /* Design Tokens - Shadows */
  --shadow-sm: 0 1px 2px 0 rgb(0 0 0 / 0.05);
  --shadow-md: 0 4px 6px -1px rgb(0 0 0 / 0.1), 0 2px 4px -2px rgb(0 0 0 / 0.1);
  --shadow-lg: 0 10px 15px -3px rgb(0 0 0 / 0.1), 0 4px 6px -4px rgb(0 0 0 / 0.1);

  /* Design Tokens - Transitions */
  --transition-fast: 150ms;
  --transition-normal: 200ms;
  --transition-slow: 300ms;

  /* Design Tokens - Spacing for sections */
  --spacing-section-sm: 3rem;
  --spacing-section-md: 4rem;
  --spacing-section-lg: 5rem;
}

/* Base styles */
body {
  font-family: var(--font-geist-sans), system-ui, sans-serif;
  color: var(--color-text-primary);
  background-color: var(--color-bg-white);
}

/* Smooth scrolling */
html {
  scroll-behavior: smooth;
}

/* Focus styles for accessibility */
*:focus-visible {
  outline: 2px solid var(--color-coral);
  outline-offset: 2px;
}
```
</task>
