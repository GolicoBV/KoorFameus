# Plan 01-04: Deploy to Vercel via GitHub

## Objective
Push the project to GitHub and deploy to Vercel with automatic deployments on every push, achieving continuous deployment within 3 minutes.

## must_haves
- Code pushed to GitHub repository
- Project deployed on Vercel
- Automatic deployments configured (push to main triggers deploy)
- Production URL accessible and working
- Deployment completes within 3 minutes
- Site displays correctly on production URL

<task id="4" depends_on="3">
Create GitHub repository and push code.

**Option A: Using GitHub CLI (if installed)**
```bash
cd c:/projects/KoorFameus
gh repo create KoorFameus --public --source=. --remote=origin --push
```

**Option B: Manual setup**
1. Go to https://github.com/new
2. Create repository named "KoorFameus"
3. Do NOT initialize with README (we have our own)
4. Copy the repository URL

Then run:
```bash
cd c:/projects/KoorFameus
git remote add origin https://github.com/YOUR_USERNAME/KoorFameus.git
git branch -M main
git push -u origin main
```

**Note:** Replace YOUR_USERNAME with actual GitHub username.
</task>

<task id="5" depends_on="4">
Deploy to Vercel.

**Option A: Using Vercel CLI**
```bash
cd c:/projects/KoorFameus
npx vercel --yes
```

Then link to production:
```bash
npx vercel --prod
```

**Option B: Via Vercel Dashboard**
1. Go to https://vercel.com/new
2. Click "Import Git Repository"
3. Select the KoorFameus repository
4. Framework: Next.js (auto-detected)
5. Click "Deploy"

**Option C: Using MCP Tool**
Use the `mcp__vercel__deploy_to_vercel` tool to deploy directly.

Vercel will auto-detect:
- Framework: Next.js
- Build Command: `npm run build` or `next build`
- Output Directory: `.next`
- Install Command: `npm install`
</task>
